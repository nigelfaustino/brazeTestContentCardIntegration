# brazeTestContentCardIntegration

This demo app is for Nigel Faustino's FY21 Q4 project. It's a basic implementation of the following Braze features:
* User analytics tracking
* User alias
* Custom Attributes
* Custom Events
* Content Cards Feed
* Filtered Content Cards Based on Input
* In-app messages

The app is built programmatically with Swift and Swift Package Manager.
